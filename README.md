A modern, responsive, and beautifully styled HTML/CSS portfolio showcasing my web development projects. This project evolved from initial bootcamp exercises into a fully styled, multi-page portfolio website.

---

## 🌟 Features

- **Modern UI**: Clean, grid-based design with subtle animations and shadow effects.
- **Responsive Layout**: Adapts perfectly to mobile, tablet, and desktop screens.
- **Projects Showcase**:
  - `movie-ranking.html` — A curated list of my top 3 favorite movies.
  - `birthday-invite.html` — A fun birthday invitation page with an embedded Google Maps link.
- **Personal Information**:
  - `about.html` — A brief introduction about myself.
  - `contact.html` — Contact details and links.
- **Semantic HTML5**: Clean and accessible markup structure.

---

## 🚀 How to Run Locally

You don't need a local server to view this portfolio.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Nejy190501/HTML-Portfolio.git
   ```
2. **Open it in your browser:**
   Double-click on `index.html` to launch the site.
